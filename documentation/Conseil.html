    class Conseil

    La classe Conseil représente un conseil. Dans le jeu, les conseils sont fournis par le système d'aide. Cette classe contient les variables d'instance suivantes :

    @listeDesIlesConcernees

    Liste des îles concernées par le conseil

    @nbIlesConcernees

    Nombre d'îles concernées par le conseil

    @texte

    Texte d'application du conseil

    @priorite

    Entier qui représente l'indice de priorité du conseil. Plus il est bas, plus le conseil est prioritaire.
